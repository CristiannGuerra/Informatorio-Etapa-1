# Selector de ID

Los selectores de ID en CSS son como identificadores únicos para elementos HTML, permitiendo aplicar estilos específicos a un solo elemento en toda la página. A diferencia de los selectores de clase que se pueden usar en varios elementos, un ID solo puede usarse una vez. Se escriben con un símbolo de almohadilla o numeral (#) seguido del nombre y se pueden combinar con selectores de tipo para una selección más precisa.

1. Ejecuta el archivo `index.html` con **LiverServer**.
2. Utilizando el **selector por ID** dale el color `rebeccapurple` al titulo de la pagina.
3. Utilizando el selector por ID coloca un `background-color : yellow` solamente al span con id `amarillo`.
